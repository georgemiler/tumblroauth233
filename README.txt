Tumblr OAuth for Tumblr API v2

Tim Gavin | tim@timgavin.name | http://timgavin.name

Example code and screencast for integrating Tumblr OAuth into your PHP project, forked from https://github.com/rchavik/TumblrOAuth

Source files: https://github.com/timgavin/tumblroauth
Screencast: http://youtu.be/xdK0dXluKIQ


- Upload the files to your server.
- Change the values in config.php to match your site.
- Add the users_oauth table to your MySQL database.
- Watch the screencast.
- Visit index.php or post.php to get things started!
